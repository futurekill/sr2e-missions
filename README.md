# Shadowrun 2E: Missions

Adventures from *Missions* (FASA 7325) for the [Shadowrun 2nd Edition system](../sr2e-foundryvtt),
as GM journals, NPC stat blocks, and maps — built to fill the downtime gaps in the
**Double Exposure** campaign.

Phase 1: **Under the Influence** and **Malpractice** (the two Seattle adventures),
plus a "Filling Double Exposure's Gaps" GM journal mapping each to a DE downtime
window. Requires the `sr2e` system. Personal table use only.
