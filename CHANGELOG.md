# Changelog

## 0.1.0 — Missions Phase 1

First release. Two complete *Shadowrun Missions* adventures as SR2E compendia:
**Under the Influence** and **Malpractice** — **23 journal entries** (scene-by-scene
GM text, handouts, legwork) and **16 cast actors** (NPCs and critters), built for
the `sr2e` system.

Not yet included: the battle maps as **Scenes** (`mi-scenes` ships as an empty
placeholder) — planned for a later release.
