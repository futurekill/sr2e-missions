# Changelog

## 0.3.0 — Cast portraits

All 14 Missions NPCs now have custom painterly portraits (square 1024px,
rotation-locked) instead of the mystery-man placeholder.

## 0.2.0 — Scenes

The `mi-scenes` pack is no longer empty: **15 placeholder battlemaps**, one per
act of *Malpractice* (8) and *Under the Influence* (7). Each is a labeled
1600×1200 grid map — swap the background for final map art; the Scene keeps its
grid and navigation name. Generated by `tools/gen-scenes.mjs`.

## 0.1.0 — Missions Phase 1

First release. Two complete *Shadowrun Missions* adventures as SR2E compendia:
**Under the Influence** and **Malpractice** — **23 journal entries** (scene-by-scene
GM text, handouts, legwork) and **16 cast actors** (NPCs and critters), built for
the `sr2e` system.

Not yet included: the battle maps as **Scenes** (`mi-scenes` ships as an empty
placeholder) — planned for a later release.
